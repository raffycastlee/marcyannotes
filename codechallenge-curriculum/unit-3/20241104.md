## Code Challenge
```js
/*
## PEDAC
**Problem**
Parse a number into individual digits and output the difference between the sum of all evens and all odds.
Number -> Number indicating the difference (as described above)

**Examples/Edge Cases:** 
2134 -> 6 (evens) - 4 (odds) == 2

**Data Structure**
- Number
- String
- Array

**Algorithm** 
1. Convert number into string.
2. Split string into array.
4. Convert each item into a number.
3. Look at each number in the array: if even, add value to sum, else subtract value from sum.
*/
function solution(n) {
    return String(n) // step 1
             .split('') // step 2
             .map(val => Number(val)) // step 3
             .reduce((diff, val) => // step 4
               (val%2 == 0) ? diff+val : diff-val, 0);
}
```
